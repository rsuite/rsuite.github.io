## Development



**First**, Fork `https://github.com/rsuite/rsuite` and clone it locally.

```zsh
git clone git@github.com:<YOUR NAME>/rsuite.git

cd rsuite
npm install
```

Start `webpack` server

```zsh
npm run dev
```

**Then**, Fork `https://github.com/rsuite/rsuite.github.io` and clone it locally.

